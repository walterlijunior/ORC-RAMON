# ORC-RAMON

**Projeto ORC-RAMON** é um projeto sem fins lucrativos. Ele foi criado com o objetivo de contribuir para o enriquecimento acadêmico, proporcionando uma ferramenta útil para a análise e interpretação de dados.

O Projeto ORC-RAMON é um script que permite ao usuário importar, tabular, tratar, analisar e interpretar dados de um arquivo Excel, e exportar os resultados para outro arquivo Excel.

Pensado inicialmente para a disciplina de Orçamento Público do curso de ciências contabéis onde o professor pedi ao aluno para baixar o balancete de despesa do municipio e iniciar as seguintes atividades:

1 - O aluno deverá utilizar o Balancete da Despesa para identificar as unidades orçamentárias, fazendo os filtros para de cada unidade orçamentária para totalizar o orçamento consignado na unidade (os valores constam na coluna dotação inicial do Balancete da Despesa). Em seguida, confrontar os dados obtidos no Balancete da Despesa com os da lei orçamentária do município. 

2 - O aluno deverá trazer dados tabulados do arquivo Balancete da Despesa do município sobre o total do orçamento aprovado pelo Poder Legislativo (Câmara Municipal) por função, programas e ações, conforme seguem: 

a) apresentar as funções que tem maior e menor percentual de orçamento aprovado; 

b) apresentar os programas com maior e menor recursos orçamentários aprovados (código e nome); 

c) apresentar a quantidade de ações; 

d) apresentar as ações com maior e menor recursos orçamentários aprovados (código e nome); 

e) apresentar o total de recursos aprovados para as ações atividade; 

f) apresentar o total de recursos aprovados para as ações projeto; 

g) apresentar o total de recursos aprovados para as ações operações especiais.

h) Citar exemplos de ações Atividade, Projeto e Operações Especiais.

## Como funciona?

1. Ao iniciar o script, será perguntado se você deseja iniciar a análise. Se a resposta for sim, o script prosseguirá com as próximas etapas.
2. Uma janela será aberta solicitando o arquivo Excel que será usado para a análise.
3. Após selecionar o arquivo, outra janela será aberta. Nesta janela, você deve indicar onde deseja salvar o arquivo resultante após a análise e também fornecer um nome para este arquivo.
4. Com tudo configurado, você pode começar a análise. O script irá orientá-lo durante todo o processo, solicitando as ações necessárias.
5. Note que acima o código mostrará todas as colunas com valores
6. Ele irá pergunta as colunas que deseja manter. O usuário pode usar de orientação as colunas que ele mostra. PAY ATTENTION: Aqui você vai inserir as colunas que deseja manter.
7. Depois, perguntará ao usuário a coluna a qual deseja agrupar.
8. Proximo passo, ele solicitará ao usuário a classificação, a ordem em que será classicificado as linhas da coluna.
9. Logo após, ele irá perguntar se já finalizou, ou seja, já trabalhou todos os dados e as colunas que deveria. Se já foi finalizado será criado o arquivo em excel, se não, ele ira retornar todo o processo e irá trabalhar as colunas que não realizou o tratamento de dados. Cada Coluna tratada, será criada uma planilah com o nome da coluna dentro do arquivo excel.
